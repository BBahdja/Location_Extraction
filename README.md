# Location_Extraction

This code enables you to obtain the latitude and longitude coordinates for specified locations based on their names.
The sample (excel file) provided here is a dataset extracted from an event-based surveillance systems, each line correspond to an article (textual data) that report one or more African Swine Fever (ASF) outbreaks.
The locations mentioned in the text were extracted, at a minimum, at the country level and at the city or department level when specified
